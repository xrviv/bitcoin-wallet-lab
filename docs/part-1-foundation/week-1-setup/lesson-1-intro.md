# Lesson 1: Course Intro

This lesson explains what we are building and how the course works.

## What you will do
- See the full course path
- Understand the project goal
- Know how to use the lesson pages

## Course path
The course has 7 parts.  
The course runs for 12 weeks.  
Each week has lessons and a quiz.  
Read [Course Outline (Learning)](/course-outline-learning/) for the full path.  
Part 7 has a short history of Android wallet stacks.  

## Project goal
You will build a Bitcoin wallet app for Android.  
You will learn Android basics first.  
You will add Bitcoin and security features later.  
You will finish with a final app in Part 7.  

## How the course works
Each lesson has a goal and steps.  
Follow the steps in order.  
Java comes first and Kotlin is optional.  
Use the quiz to check your understanding.  

## How to use lesson pages
Read the page top to bottom.  
Click Next to move forward.  
Go back if you need a review.
