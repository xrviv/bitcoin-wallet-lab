# Lesson 3: Basic Project Setup

This lesson creates your first empty Android app.

## What you will do
- Install Android Studio if you have not yet.
- Create a new empty project.
- Run the app on a device or emulator.

## Steps
1. Open Android Studio.
2. Click "New Project."
3. Choose an Empty Activity template.
4. Name the project and pick a save folder.
5. Click "Run" and wait for the app to launch.

## If something fails
- Check that you have the Android SDK installed.
- Check that your emulator or device is connected.
