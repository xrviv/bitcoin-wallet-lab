# Lesson 2: Tools and Setup

We list the tools you need and how to confirm they work.

## What you will install
- Java and Android tools
- Android Studio
- A working emulator or device

There are many paths to making an Android app.  
The earliest Bitcoin Android wallet was coded in Java with the native Android SDK.  

## Other build paths
These are common paths used today.  
- Native Android with Kotlin.  
- Native Android with Java.  
- Flutter with Dart.  
- React Native with JavaScript or TypeScript.  
- Kotlin Multiplatform for shared logic with native UI.  
- .NET MAUI with C#.  
- Xamarin with C# (legacy, still in the wild).  
- Unity with C#.  
- Web app in a WebView wrapper.  
- Progressive Web App in a browser.  
- Compose Multiplatform (experimental for Android and desktop).  
- Qt with C++ or QML.  
These are valid paths.  
This course uses Android Studio to stay close to the platform and tools.  
It is the official IDE for Android.  
It gives direct access to the Android SDK and emulator.  
It also keeps the build steps consistent for learning and reproducibility.
