# Code of Conduct

We want this project to be safe and welcoming for everyone.

## Be kind
- Use respectful language.
- Assume good intent.
- Help others learn.

## Not allowed
- Harassment or threats
- Hate speech or discrimination
- Sharing private information without permission

## Reporting problems
If something goes wrong, open an issue and tag the maintainer.

Thank you for helping keep this project friendly.
