# Bitcoin Wallet Android Course

Learn to build a Bitcoin wallet for Android, step by step.

## What this repository is
- Course lessons and documents
- Code snapshots for each part
- Exercises and solutions
- Reference resources
- Reproducibility notes and checks across the course

## Quick start
1. Read the course outline in `COURSE_OUTLINE.md`.
2. Open the docs site files in `docs/`.

## Folder map
- `docs/` — course lessons and website pages
- `code/` — code snapshots after each part
- `exercises/` — practice tasks
- `solutions/` — answer keys
- `resources/` — cheatsheets and references
